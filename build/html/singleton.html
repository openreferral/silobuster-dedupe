singleton module¶

SingletonMeta turns a class into a Singleton class. There are numerous methods on StackOverflow to accomplish this. As of Python 3.6, all methods have nuances and have unexpected behavior except for this method.

USAGE:

class MyClass(metaclass=SingletonMeta):

pass

NOTES: MetaClass is a class that inherits type. Type is called when an object is created. class MyObject is equivalent to type(MyObject, bases, args) ..

Here a metaclass returns the instance that has already been created. This method is preferred over the “new” dunder method returning the instance. In that method, properties do not behave correctly.
